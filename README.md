## 📖 Project Overview
ShopWave-Starter is a fundamental Spring Boot enterprise application demonstrating core backend architecture, including Inversion of Control (IoC), Dependency Injection, and layered application design (Controller, Service, Repository). It utilizes Spring Data JPA with an in-memory H2 database for rapid development and testing.
